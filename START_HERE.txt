Welcome! Follow the steps below for your operating system.
The whole setup takes about 5 minutes.

You will need:
  - The Aruni password (ask whoever sent you this file)
  - Internet connection

----------------------------------------------------------------------
  MAC / LINUX
----------------------------------------------------------------------

1. Open Terminal

2. Go to this folder:
      cd ~/Downloads/aruni

3. Run setup:
      bash setup_new_machine.sh

4. Enter the Aruni password when asked.

5. Done! The script will guide you through everything.

----------------------------------------------------------------------
  WINDOWS
----------------------------------------------------------------------

1. Open Command Prompt (search "cmd" in Start menu)

2. Go to this folder:
      cd %USERPROFILE%\Downloads\aruni

3. Run setup:
      setup_new_machine.bat

4. Enter the Aruni password when asked.

5. Done! The script will guide you through everything.
